# Parts
1X Seeed XIAO RP2040,
9X 1N4148 Diodes,
9X MX-Style switches,
1X EC11 Rotary encoder,
1X 0.91 inch OLED display,
9X DSA Keycaps,
4X M3x16mm screws,
4X M3x5mx4mm Heatset Inserts
